# sabinshamsul.dev

Personal portfolio website for **Syahmi Shamsul** — a Computer Science student at Taylor's University specialising in Data Science & AI.

## Features

- Dark / Light mode toggle with localStorage persistence
- Fully responsive design (desktop, tablet, mobile)
- GitHub-inspired UI and color palette
- Animated scroll reveals and hero entrance
- Project showcase with live demo links
- Interactive code card in hero section

## Tech Stack

- HTML / CSS / JavaScript (single-page, no frameworks)
- CSS custom properties for theming
- Google Fonts (Syne, Space Mono, DM Sans)
- Hosted on GitHub Pages

## Projects Showcased

- **MediVision** — Smart Patient Flow Platform (Capstone Project)
- **Mental Health Sentiment Analysis** — Deep Learning text classification
- **Bone Fracture Detection** — CNN-based X-ray analysis
- **Loan Default Prediction** — ML risk assessment model
